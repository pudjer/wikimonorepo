сайт где пользователи смогут обучаться по строго структурированной информации в виде dag

пользователь сможет отслеживать свой прогресс по изученым подстатьям

рекомендации строятся по общему рейтингу статей и личной релевантности

общий рейтинг строится из метрик самой статьи, а так же того, насколько от нее зависим глобальный dag(наследственность рейтинга родителей)

релевантность строится как по общим рекомендательным алгоритмам так и по взаимодействиям пользователя со смежными статьями

смежность статей определяется путями в dag(идеальный случай: выучены все родители - рекомендован ребенок)

структура адаптирована под последующие интеграции ии ассистентов:
    навигация(возможность поиска для агента и предоставление информации для родителей):
        помощь в поиске релевантных статей
        переключения контекста при вопросах о зависимостях и возможность возврата(краткое изложение текущего контекста при переключение в другой)

    хранение контекста пользователя об объяснениях для каждой статьи



    объяснения так же могут учитываться в релевантности

    ии ассистент в первую очередь ссылается на подграф статьи

    возможность настройки режима RAG ии ассистента:
        1 строгий: поиск и ссылки только по подграфам и исключение не содержащейся в нем информации
        2 пользовательский: ии ассистент в контексте одной статьи может собирать метрики для авторов к содержанию каких статей не содержащихся в родителях он ссылается в объяснении по процентах от пользователей
